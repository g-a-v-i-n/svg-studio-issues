# SVG Studio

![SVG Studio interface](https://github.com/g-a-v-i-n/svg-studio-issues/blob/main/svg-studio.png)

SVG Studio is a browser-based tool for developers and designers who work with SVG.

Use SVG Studio to:

- Optimize SVG assets for web performance
- Understand, debug and fix SVG rendering issues
- Convert and clean up SVGs originating from design tools
- Tinker with hard-to-understand parts of SVG with immediate visual feedback
- Prepare SVG icons and illustrations for production
